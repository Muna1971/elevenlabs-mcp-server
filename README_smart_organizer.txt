================================================================================
          اداة التنظيم الذكي لسطح المكتب - Smart Desktop Organizer
================================================================================

[الوصف]
--------
هذه اداة Python تقوم بتحليل ملفات سطح المكتب وتصنيفها تلقائيا في مجلدات
بناء على محتوى الملفات واسمائها.

================================================================================

[المتطلبات]
-----------
1. Python 3.6 او اعلى
   - للتحقق من الاصدار: python --version

2. مكتبات اختيارية (موصى بها):
   - PyPDF2: لقراءة ملفات PDF
   - python-docx: لقراءة ملفات Word

================================================================================

[التثبيت]
---------
الخطوة 1: تحميل الملف
   - احفظ ملف smart_organizer.py في اي مكان على جهازك

الخطوة 2: تثبيت المكتبات (اختياري لكن موصى به)
   افتح موجه الاوامر (Command Prompt) او Terminal واكتب:

   pip install PyPDF2 python-docx

   او اذا كان لديك pip3:

   pip3 install PyPDF2 python-docx

================================================================================

[التخصيص - مهم جدا!]
---------------------
قبل استخدام الاداة، يجب تعديل قسم التخصيص في الملف:

1. افتح ملف smart_organizer.py بمحرر نصوص (Notepad, VS Code, etc.)

2. ابحث عن قسم التخصيص (CUSTOMIZATION)

3. عدل الاعدادات التالية:

   أ) اسماء التصنيفات الرئيسية:
      ----------------------------------------
      CATEGORY_1_NAME = "العمل"           # او "Work"
      CATEGORY_2_NAME = "الدراسة"         # او "Study"
      CATEGORY_3_NAME = "مشاريع شخصية"    # او "Personal"

   ب) الكلمات المفتاحية لكل تصنيف:
      ----------------------------------------
      مثال للتصنيف الاول (العمل):

      CATEGORY_1_KEYWORDS = {
          "general": [
              "عمل", "اجتماع", "تقرير", "مشروع",
              "work", "meeting", "report", "project",
          ],
          "المالية": [
              "فاتورة", "ميزانية", "محاسبة",
              "invoice", "budget", "accounting",
          ],
          "الموارد البشرية": [
              "توظيف", "اجازة", "راتب",
              "hiring", "vacation", "salary",
          ],
      }

      مثال للتصنيف الثاني (الدراسة):

      CATEGORY_2_KEYWORDS = [
          "جامعة", "محاضرة", "امتحان", "واجب",
          "university", "lecture", "exam", "homework",
      ]

   ج) التصنيفات الفرعية (اختياري):
      ----------------------------------------
      CATEGORY_1_SUBCATEGORIES = [
          "المالية",
          "الموارد البشرية",
          "التسويق",
      ]

      SUBCATEGORY_FOLDER_NAME = "الاقسام"

   د) مسارات مخصصة (اذا كان سطح المكتب في مكان غير تقليدي):
      ----------------------------------------
      CUSTOM_DESKTOP_PATHS = [
          Path.home() / "OneDrive - اسم الشركة" / "Desktop",
      ]

================================================================================

[طريقة التشغيل]
---------------
الخطوة 1: العرض التجريبي (بدون نقل الملفات)
   ----------------------------------------
   افتح موجه الاوامر في مجلد الملف واكتب:

   python smart_organizer.py

   سيعرض لك كيف سيتم تصنيف الملفات دون نقلها فعليا.

الخطوة 2: التنفيذ الفعلي (نقل الملفات)
   ----------------------------------------
   بعد التاكد من صحة التصنيف، اكتب:

   python smart_organizer.py --run

   سيتم انشاء المجلدات ونقل الملفات تلقائيا.

[ملاحظات مهمة]
--------------
1. الاداة تقرا محتوى الملفات لتصنيفها بدقة اكبر
2. بدون المكتبات الاختيارية، ستعتمد على اسم الملف فقط
3. الملفات التي لا تطابق اي تصنيف توضع في مجلد "غير مصنف"
4. دائما استخدم العرض التجريبي اولا قبل التنفيذ الفعلي
5. الاداة لا تحذف الملفات - فقط تنقلها الى مجلدات

[حل المشاكل]
------------
مشكلة: "python is not recognized"
   الحل: تاكد من تثبيت Python واضافته الى PATH

مشكلة: لم يتم العثور على سطح المكتب
   الحل: اضف المسار في CUSTOM_DESKTOP_PATHS

مشكلة: الملفات لا تصنف بشكل صحيح
   الحل: اضف كلمات مفتاحية اكثر دقة
